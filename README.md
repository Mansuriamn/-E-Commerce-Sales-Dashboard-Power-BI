## 📌 Project Overview

Businesses generate thousands of sales records every day, but raw data alone doesn't drive business growth. Decision-makers need meaningful insights to understand performance, identify trends, and make informed decisions.

This project demonstrates how Power BI can transform raw e-commerce sales data into an interactive dashboard that enables users to monitor business performance, evaluate profitability, and explore customer purchasing behavior.

The dashboard focuses on answering important business questions such as:

- Which months generate the highest profit?
- Which product categories contribute the most sales?
- Which payment methods are most commonly used?
- Which states generate the highest revenue?
- Who are the top customers?
- How do sales vary across different quarters?
